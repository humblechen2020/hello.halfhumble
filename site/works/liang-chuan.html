良全 — 品牌識別優化設計Liang Quan — Brand Identity Optimization

TBrand Identity PCase Study DHanpo Chen Y2026 類別品牌識別 專案性質實例專案 設計品牌設計:陳翰博 年份2026

Liang Quan is a third-generation, family-run ready-mix concrete plant in central Taiwan. The brand already had a identity in place, but it had grown inconsistent across two decades of trucks, bags, and stationery. The 2026 brief was not to start over, but to optimize — sharpen the mark, tighten the system, and make it hold up everywhere it's seen.

良全預拌混凝土是傳承三代的中部家族企業。品牌原本就有一套識別，但歷經二十年累積，在貨車、包裝袋、文具上已經出現不一致。2026 年的任務不是從零開始，而是優化——把標誌收斂得更精確，把系統收得更緊，讓識別在每一個被看見的地方都站得住。

02 — Problem

專案背景與痛點梳理Background & Problem

良全的識別不是沒有，而是用了太久、用得太分散——每個部門各自詮釋同一套規範，結果是哪裡都看得到良全，卻沒有一個地方看起來是同一個良全。Liang Quan's identity wasn't missing — it had simply been used too long, in too scattered a way. Each department interpreted the same guidelines on its own, so Liang Quan ended up visible everywhere, yet looking like the same Liang Quan nowhere.

標誌在不同年代的貨車、包裝袋、名片上有著微妙的版本差異；色彩在印刷與招牌上也各自漂移。業主真正想被解決的問題，不是「我們需要一個新識別」，而是「我們的識別已經不準了，要怎麼收回來」。這是一次優化，不是一次重建。

The mark carried subtle version differences across trucks, bags, and business cards from different eras; colors had drifted across print and signage too. What the client actually wanted solved wasn't "we need a new identity," but "our identity has drifted out of accuracy — how do we pull it back in." This was an optimization, not a rebuild.

這次的任務，是重新梳理良全既有的識別語彙，把標誌、色彩、字級收斂成一套精確、可被任何人正確套用的系統，再回頭檢查每一個應用端是否都對齊了同一套規則。

The task was to re-organize Liang Quan's existing identity vocabulary — tightening the mark, color, and typography into one precise system that anyone could apply correctly — then go back and check whether every application actually aligned with that same set of rules.

03 — Strategy

設計策略與系統梳理Design Strategy & System Audit

先盤點，再優化Audit First, Then Optimize

把這些年所有實際在用的識別應用都收回來比對，找出哪些版本是對的、哪些是走偏的，再回頭定義一套唯一、精確的標誌比例與配色數值。

We gathered every identity application actually in use over the years and compared them, identifying which versions were correct and which had drifted, then defined a single, precise set of logo proportions and color values.

識別系統的執行策略Identity System Execution Strategy

標誌重新定義安全範圍與最小可讀尺寸，色彩系統收斂成固定的印刷與數位色號，字級規則訂出標題與資訊層級的對應關係。系統文件本身也重新設計成更容易被現場人員直接查閱、直接套用的格式，減少日後再次走偏的機會。

The mark was given a redefined safe zone and minimum legible size, the color system was tightened into fixed print and digital color values, and typography rules established a clear correspondence between heading and information hierarchy. The system documentation itself was redesigned into a format on-site staff could check and apply directly, reducing the chance of future drift.

04 — Design Details

設計深化／從系統規範到現場應用Design Details — From System Guidelines to On-Site Application

∞標誌結構與比例優化Mark Structure & Proportion Optimization

重新定義標誌的幾何構成、安全範圍與最小可讀尺寸，確保從名片到貨車車身，任何尺度下都是同一個精確比例，不再因為手動縮放而失真。

We redefined the mark's geometric construction, safe zone, and minimum legible size, ensuring the exact same proportion holds true at any scale, from business cards to truck bodies, no longer distorted by manual resizing.

∞色彩與字級系統校正Color & Typography System Correction

把長年累積的印刷色號落差收斂成固定數值，字級系統也重新訂出標題、資訊、註記三個層級的對應關係，讓任何人套用都能得到一致的結果。

Years of accumulated print-color drift were tightened into fixed values, and the typography system was redefined across three levels — heading, information, and annotation — so anyone applying it gets a consistent result.

∞應用端規範與檢核Application Guidelines & Audit

逐一檢核貨車車身、包裝袋、名片、招牌等既有應用，標出走偏的版本並提供修正後的標準檔案，讓整套識別在所有現場真正落地成同一個樣子。

We audited existing applications one by one — truck bodies, bags, business cards, signage — flagging drifted versions and supplying corrected standard files, so the whole identity actually lands as the same thing everywhere it's seen.

05 — Conclusion

同一套識別，第一次真正一致One Identity, Truly Consistent for the First Time

這次的識別優化，把良全二十年累積下來的版本落差收回到同一套精確系統。從貨車到名片，識別第一次在每一個應用端都站得住、也站得一致——優化不是重新發明一個品牌，而是讓既有的品牌終於說得清楚自己是誰。

This identity optimization pulled Liang Quan's twenty years of accumulated version drift back into one precise system. From trucks to business cards, the identity holds up — and holds together — at every application for the first time. Optimization isn't reinventing a brand; it's letting an existing brand finally say clearly who it is.

重點回顧
Key Points

  • 識別稽核盤點：收集所有現存應用版本，比對出走偏的細節，找出唯一正確的標準。
  • Identity audit: all existing application versions gathered and compared to identify drifted details and the one correct standard.
  • 系統校正：重新定義標誌比例、色彩數值與字級階層，收斂成一套精確規範。
  • System correction: mark proportions, color values, and typography hierarchy redefined and tightened into one precise set of guidelines.
  • 應用端檢核：逐一修正貨車、包裝、文具等既有應用，讓識別真正落地一致。
  • Application audit: existing applications — trucks, packaging, stationery — corrected one by one so the identity truly lands consistently.
